# サイト構造設計

## URL構造
```
https://habitengineer.com/
├── /                           ← トップページ
├── /services/                  ← サービス詳細
├── /results/                   ← 実績一覧
│   ├── /results/cases/         ← 導入事例詳細
│   └── /results/products/      ← 開発製品詳細
├── /about/                     ← 事業者情報
└── /blog/                      ← ブログ
    ├── /blog/category/ai/      ← カテゴリ別
    ├── /blog/category/notion/
    └── /blog/2025/01/01/post-title/
```

## ナビゲーション構造
```
HabitEngineer (ロゴ)
├── とは (/)
├── サービス (/services/)
├── 実績 (/results/)
│   ├── 導入事例 (/results/cases/)
│   └── 開発製品 (/results/products/)
├── 事業者情報 (/about/)
└── ブログ (/blog/)
```

## パンくずリスト
- トップ > サービス
- トップ > 実績 > 導入事例
- トップ > 実績 > 開発製品
- トップ > 事業者情報
- トップ > ブログ > カテゴリ > 記事タイトル
