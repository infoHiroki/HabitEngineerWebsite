# デザインシステム

## カラーパレット
```css
/* メインカラー */
--sakura-pink: #FFC0CB;      /* 桜色 */
--sakura-light: #FFE4E6;     /* 薄い桜色 */
--sakura-dark: #E6A8B8;      /* 濃い桜色 */

/* ベースカラー */
--white: #FFFFFF;            /* 背景メイン */
--black: #333333;            /* テキストメイン */
--gray: #666666;             /* テキストサブ */
--gray-light: #F5F5F5;       /* 背景サブ */
```

## タイポグラフィ
```css
/* フォントファミリー */
font-family: 'Noto Sans JP', 'Hiragino Kaku Gothic ProN', sans-serif;

/* フォントサイズ */
--h1: 32px;     /* ページタイトル */
--h2: 24px;     /* セクションタイトル */
--h3: 20px;     /* サブタイトル */
--body: 16px;   /* 本文 */
--small: 14px;  /* 注釈 */
```

## ボタンスタイル
```css
/* プライマリボタン */
background: var(--sakura-pink);
color: var(--white);
border-radius: 8px;
padding: 12px 24px;

/* セカンダリボタン */
background: var(--white);
color: var(--sakura-pink);
border: 2px solid var(--sakura-pink);
border-radius: 8px;
padding: 12px 24px;
```
