# コンポーネント設計

## カードコンポーネント

### 基本カード
```html
<div class="card">
  <div class="card-header">
    <h3 class="card-title">タイトル</h3>
  </div>
  <div class="card-body">
    <p class="card-text">コンテンツ</p>
  </div>
  <div class="card-footer">
    <a href="#" class="btn btn-primary">詳しく見る</a>
  </div>
</div>
```

## ボタンコンポーネント

### プライマリボタン
```html
<a href="#" class="btn btn-primary">
  <span class="btn-text">サービスを見る</span>
</a>
```

### セカンダリボタン
```html
<a href="#" class="btn btn-outline">
  <span class="btn-text">詳しく見る</span>
</a>
```

### CTAボタン（大）
```html
<a href="mailto:service@habitengineer.com" class="btn btn-cta">
  <span class="btn-text">お問い合わせ</span>
  <span class="btn-subtext">service@habitengineer.com</span>
</a>
```

## ナビゲーションコンポーネント

### メインナビゲーション
```html
<nav class="main-nav">
  <ul class="nav-list">
    <li class="nav-item">
      <a href="/" class="nav-link">とは</a>
    </li>
    <li class="nav-item nav-item-dropdown">
      <a href="/results/" class="nav-link">実績</a>
      <ul class="nav-dropdown">
        <li><a href="/results/cases/">導入事例</a></li>
        <li><a href="/results/products/">開発製品</a></li>
      </ul>
    </li>
  </ul>
</nav>
```

### パンくずナビゲーション
```html
<nav class="breadcrumb">
  <ol class="breadcrumb-list">
    <li class="breadcrumb-item">
      <a href="/">トップ</a>
    </li>
    <li class="breadcrumb-item">
      <a href="/results/">実績</a>
    </li>
    <li class="breadcrumb-item active">
      導入事例
    </li>
  </ol>
</nav>
```
